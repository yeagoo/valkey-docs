## Valkey란

Valkey는 2024년 3월 Redis 7.2.4에서 분기된 BSD-3-Clause 라이선스의 인메모리 키-값 저장소입니다. Linux Foundation이 관리하며, AWS, Google Cloud, Oracle 등이 후원합니다.
